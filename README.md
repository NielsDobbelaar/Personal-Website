## Getting Started

### Prerequisites

1. have node(v20.15.0) and npm(v10.7.0) installed

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/NielsDobbelaar/Personal-Website
   ```
2. Install the dependencies
   ```sh
   npm i
   ```
3. Start the development server
   ```sh
   npm run dev
   ```
